### Run project locally
1. Clone git repository
2. Install node.js
3. Open local repository in terminal
4. Install all packages 
```javascript
npm install
```
5. Install gulp globally
```javascript
npm install gulp -g
```

### Run Gulp scripts
**For dev mode**
```javascript
gulp
```

**For prod mode**
```javascript
gulp prod
```
